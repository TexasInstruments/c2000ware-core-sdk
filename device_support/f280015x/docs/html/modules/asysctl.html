ASysCtl Module¶

The ASysCtl or Analog System Control driver provides functions to enable, disable and lock the temperature sensor on the device. It will also provide additional functionality if available for that device.

void ASysCtl_enableCMPSSExternalDAC(uint32_t select)¶

Enables the CMPSSx.DACL output to be routed to a pin.

This function disables CMPSSx.COMPL and the associated DAC will act as a general purpose DAC with 11 bit resolution. The parameter

select can be OR of the following values:

  • ASYSCTL_CMPSS1_DACL

Parameters

  • select: is CMPSS instance

Note

In this case, all other CMPSS module functionality is not useable, including the high DAC, both comparators, ramp generation, and the digital filters.

Return

None.

void ASysCtl_disableCMPSSExternalDAC(uint32_t select)¶

Disbales the CMPSSx.DACL output to be routed to a pin.

This function enables CMPSSx.COMPL and the associated DAC will act as reference for comparator. The parameter

select can be OR of the following values:

  • ASYSCTL_CMPSS1_DACL

Parameters

  • select: is CMPSS instance

Return

None.
