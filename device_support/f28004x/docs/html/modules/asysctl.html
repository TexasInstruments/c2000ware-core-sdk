ASysCtl Module¶

The ASysCtl or Analog System Control driver provides functions to enable, disable and lock the temperature sensor on the device. It will also provide additional functionality if available for that device.

bool ASysCtl_getInductorFaultStatus(void)¶

ASysCtl_getInductorFaultStatus & ASysCtl_getSwitchSequenceStatus used for function ASysCtl_enableDCDC().

Gets the inductor status.

This function returns the inductor status.

Return

Return value true indicates that the external inductor connected to DC-DC is functional. Return value of false indicates it is faulty or not connected.

bool ASysCtl_getSwitchSequenceStatus(void)¶

Gets the Switch Sequence Status.

This function returns the Switch Sequence Status.

Return

Return value false indicates that the switch to DC-DC is not complete. Return value of true indicates it is complete.

bool ASysCtl_enableDCDC(void)¶

Enable DC-DC.

This function enables the DC-DC and checks if the enable was successful when switching from the VREG to the DC-DC .

Return

Return value true indicates that the DC-DC enable was successful when switching from the VREG to the DC-DC . Return value of false indicates that the DC-DC enable with switch failed.

Note

The internal DC-DC regulator is disabled by default. To use this supply, the device must power up initially with the internal LDO (VREG) and then transition to the internal DC-DC regulator.
