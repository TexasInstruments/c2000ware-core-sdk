CPU Module¶

The cpu API includes a header file with basic functions to read and set some core registers like enabling or disabling interrupts, reading or setting PRIMASK, BASEPRI registers etc.

group cpu_api

The module is used for configuring Cortex-M core registers.

Functions

uint32_t CPU_setFAULTMASK(void)¶

Returns the state of FAULTMASK register on entry and disable all interrupts except NMI.

This function is wrapper function for the CPSID instruction. It returns the state of FAULTMASK on entry and disable all interrupts except NMI.

Return

Returns the state of FAULTMASK register.

uint32_t CPU_getFAULTMASK(void)¶

Returns the state of FAULTMASK register

This function is wrapper function for returning the state of FAULTMASK register(indicating whether interrupts except NMI are enabled or disabled).

Return

Returns the state of FAULTMASK register.

uint32_t CPU_clearFAULTMASK(void)¶

Returns the state of FAULTMASK register and enable the interrupts

This function is wrapper function for returning the state of FAULTMASK register and enabling the interrupts.

Return

Returns the state of FAULTMASK register.

void CPU_setBASEPRI(uint32_t basePriority)¶

Writes the BASEPRI register

This function is wrapper function for writing the BASEPRI register. MSB 3-bits are enabled for setting the priority and non-implemented low-order bits read as zero and ignore writes. To set the base priority of 0x2U, the param

basePriority to be passed should be 0x20U.
Parameters

  • basePriority: is the value to be set

Return

None.
