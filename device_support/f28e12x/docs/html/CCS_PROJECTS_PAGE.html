Build Program

  • Following are the options to build and load an example program
    • Build and Load together in a single click
      • Righ click on the project and select 'Debug Project (F5)' option
        Debug CCS project
      • It performs the following sequence
        • Builds the project
        • Launches the Target Configuration based on the 'ccxml' file included in projectspec
        • Connects to the target core
        • Loads the program onto the target core
    • Build and Load separately
      • Build
        • Right click on the project and select either 'Build Project(s)' or 'Rebuild Project(s)' option
          Build CCS project
        • Upon successful build, example program is generated
        • Launch the CCS Debug Session
        • Connect to the target core
        • Load the program
