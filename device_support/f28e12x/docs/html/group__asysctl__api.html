◆ ASysCtl_enableCMPSSExternalDAC()

static void ASysCtl_enableCMPSSExternalDAC ( uint32_t  select)
inlinestatic

Enables the CMPSSx.DACL output to be routed to a pin.

Parameters
selectis CMPSS instance

This function disables CMPSSx.COMPL and the associated DAC will act as a general purpose DAC with 11 bit resolution. The parameter select can be OR of the following values:

  • ASYSCTL_CMPSS3_DACL
Note
In this case, all other CMPSS module functionality is not useable, including the high DAC, both comparators, ramp generation, and the digital filters.
Returns
None.
