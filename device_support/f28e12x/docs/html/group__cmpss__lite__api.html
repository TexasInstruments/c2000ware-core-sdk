◆ CMPSSLITE_configDAC()

static void CMPSSLITE_configDAC ( uint32_t  base,
uint16_t  config 
)
inlinestatic

Sets the configuration for the internal comparator DACs.

Parameters
baseis the base address of the CMPSSLITE module.
configis the configuration of the internal DAC.

This function configures the comparator's internal DAC. The config parameter configures the DAC value load trigger event between SYSCLK and EPWMSYNCPER i.e. the CMPSS_DACVAL_xxx value.

The CMPSS_DACVAL_xxx term can take on the following values to specify when the DAC value is loaded from its shadow register:

  • CMPSS_DACVAL_SYSCLK - Value register updated on system clock.
  • CMPSS_DACVAL_PWMSYNC - Value register updated on PWM sync.
Returns
None.
