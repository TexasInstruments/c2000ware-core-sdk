inlinestatic

Set the analog voltage reference selection to internal.

Parameters
referenceis the analog reference.

The parameter reference can be a combination of the following values:

  • ASYSCTL_ANAREF_INTREF_ADCA
  • ASYSCTL_ANAREF_INTREF_ADCB
  • ASYSCTL_ANAREF_INTREF_ADCC
  • ASYSCTL_ANAREF_INTREF_ADCD
  • ASYSCTL_ANAREF_INTREF_ADCE
Note
If any one ADC chooses INTREF then VREFHI option will become invalid for all other ADCs.

None.
