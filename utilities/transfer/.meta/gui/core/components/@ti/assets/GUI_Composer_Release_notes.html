Device Support

GUIComposer2 Beta 1.6 ships with support for the following devices built in:
  • MSP430F2617
  • MSP430F5529
  • MSP430FR4133
  • MSP430FR5969
  • MSP430FR6989
  • MSP430G2452
  • MSP430G2553
  • MSP430P401R
  • TMS320F28377D

Known Issues

Here are the known issues with this release.
  • In the Serial IO Demo project template, the ti-widget-serialio component always has the following attributes set in the gist:
    initcomplete devicename="MSP430F2617" boardname="MSP430F2617".
    Workaround: the index.js file for the project overrides these settings programmatically.
